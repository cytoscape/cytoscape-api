This document represents the API specification for Cytoscape 3.0 using a Java Swing front-end.

Getting Started

Resources for getting started with Cytoscape App development:
  • Cytoscape App Ladder (overview of setting up development environment)
  • Introduction to Cytoscape App Development (tutorial for getting started using the API)
  • Cytoscape 3 App Cookbook (API usage examples)
  • Cytoscape App Samples (github repo of example Apps)
  • cytoscape-app-dev (Google Group for asking questions about Cytoscape App development)

Cytoscape 3 is build on the OSGi framework. The Cytoscape API is divided into several bundles, each of which requires its own maven dependency. Use the tabs below to see which packages are contained within each bundle.
